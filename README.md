#  Finger Counter - Hand Gesture Recognition

یک برنامه تشخیص تعداد انگشت‌های دست با استفاده از دوربین وب‌کم، پیاده‌سازی شده با **OpenCV** و **MediaPipe**. این پروژه نقاط کلیدی دست را شناسایی کرده و تعداد انگشت‌های باز شده را به‌صورت لحظه‌ای شمارش و روی صفحه نمایش می‌دهد.

## ✨ ویژگی‌ها

- تشخیص **۲۱ نقطه کلیدی** دست با دقت بالا
- شمارش تعداد انگشت‌های باز شده در هر دو دست (چپ و راست)
- نمایش لحظه‌ای تعداد انگشت‌ها روی ویدئوی دوربین
- نمایش نقاط و اتصالات دست برای بازخورد بصری
- تشخیص خودکار دست چپ و راست
- اجرای **بلادرنگ (Real-Time)** با نرخ فریم بالا

## 🧠 نحوه عملکرد

1. **دریافت تصویر** از دوربین وب‌کم با OpenCV
2. **تشخیص دست** با استفاده از ماژول Hands کتابخانه MediaPipe
3. **استخراج نقاط کلیدی** (۲۱ نقطه مختصات x, y, z)
4. **تحلیل زاویه و موقعیت** هر انگشت برای تشخیص باز یا بسته بودن
5. **شمارش** انگشت‌های باز شده
6. **نمایش** تعداد روی صفحه ویدئو

## 🛠️ تکنولوژی‌ها

| کتابخانه | کاربرد |
|----------|--------|
| **OpenCV** | پردازش تصویر و نمایش ویدئو |
| **MediaPipe** | تشخیص نقاط کلیدی دست |
| **NumPy** | محاسبات ریاضی و آرایه‌ها |

🧪 نمونه کد (بخش تشخیص انگشت)


python
def count_fingers(hand_landmarks, image_shape):
    fingers = []
    # تشخیص باز بودن انگشت‌ها با مقایسه مختصات
    # انگشت شست
    if hand_landmarks.landmark[4].x < hand_landmarks.landmark[3].x:
        fingers.append(1)
    else:
        fingers.append(0)
    # ۴ انگشت دیگر
    for i in range(8, 21, 4):
        if hand_landmarks.landmark[i].y < hand_landmarks.landmark[i-2].y:
            fingers.append(1)
        else:
            fingers.append(0)
    return sum(fingers)
